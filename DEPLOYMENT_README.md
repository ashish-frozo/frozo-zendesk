# EscalateSafe Deployment Files

This directory contains Railway deployment configuration.

## Quick Deployment

```bash
# 1. Make deployment script executable
chmod +x deploy-railway.sh

# 2. Run preparation script
./deploy-railway.sh

# 3. Push to GitHub
git add .
git commit -m "Add Railway deployment config"
git push origin main

# 4. Deploy via Railway
# Follow DEPLOYMENT.md for step-by-step guide
```

## Files

- **Procfile** - Railway service definitions (web + worker)
- **railway.toml** - Railway build configuration
- **Aptfile** - System dependencies (Tesseract OCR, Poppler)
- **runtime.txt** - Python version specification
- **deploy-railway.sh** - Automated deployment prep script
- **DEPLOYMENT.md** - Complete step-by-step deployment guide

## Environment Variables

See `.env.railway` (generated by deploy-railway.sh) or `.env.example` for complete list.

**Required:**
- Database: Auto-provided by Railway PostgreSQL plugin
- Redis: Auto-provided by Railway Redis plugin
- S3: Cloudflare R2 or AWS S3 credentials
- APIs: Zendesk, Jira, Slack, OpenAI keys
